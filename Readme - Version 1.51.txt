	a TF2 HUD by Bing Soy

A clean & simple HUD. Made with both gameplay & fragmovies in mind. More customizations to come.

+ Customize your HP icon: go to "../BingSoyHUD/materials/hud/Custom HP Icon Skins", copy "health_bg.vtf & health_color.vtf" in any one of the folders there & replace the textures in the "hud" folder.

+ 1440p (intended look) & non-1440p (resolution compatiblity) font customizations. Go to "../BingSoyHUD/resource", replace files in the "scheme" folder with the files in one of the "#scheme" folders. This method may change later.
By default, the non-1440p font scheme files will be used so the font works for most people.

➤ See mod page for more info:
https://gamebanana.com/mods/398210
➤ Github link for more frequent updates (less stable):
https://github.com/BingSoy/BingSoyHUD
